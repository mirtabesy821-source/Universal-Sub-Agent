使用指南 / Quick Start

  1. 选择厂商并填入对应的 API Key，点击"保存设置"。
  2. 打开任意网页，用鼠标划选一段文字。
  3. 选区旁会弹出"🔍 解释"按钮，点击展开对话窗口。
  4. 在输入框中提问，AI 会结合选中文本和页面上下文进行解答。
  5. 支持多轮对话、Markdown 排版和 LaTeX 数学公式渲染。
