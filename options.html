Universal Sub-Agent

在任何网页上划选文字，即可获得 AI 局部解答

选择你要使用的 AI 服务商。所有厂商均兼容 OpenAI 协议。
→ 前往获取 API Key
Key 仅存储在本地 chrome.storage，不会上传到任何服务器。
可填写厂商支持的其它模型名称，如 deepseek-reasoner、qwen-plus 等。

使用指南 / Quick Start

  1. 选择厂商并填入对应的 API Key，点击"保存设置"。
  2. 打开任意网页，用鼠标划选一段文字。
  3. 选区旁会弹出"🔍 解释"按钮，点击展开对话窗口。
  4. 在输入框中提问，AI 会结合选中文本和页面上下文进行解答。
  5. 支持多轮对话、Markdown 排版和 LaTeX 数学公式渲染。
