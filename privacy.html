Universal Sub-Agent — 隐私政策

最后更新：2026-07-05

1. 数据收集与存储

本扩展 不收集、不存储、不传输任何用户个人身份数据。所有用户配置（包括 API Key、选择的厂商、自定义模型名称）仅存储在用户本地的 chrome.storage.local 中，开发者无法访问这些信息。

2. 权限说明

  • activeTab / scripting — 仅在用户主动点击扩展图标或划选文字时，向当前标签页注入内容脚本以显示解释按钮和浮动窗口。
  • storage — 用于在本地存储用户的 API Key、厂商选择和模型配置。
  • host_permissions: <all_urls> — 使扩展能够在任意网页上工作。页面内容仅在用户主动划选文字并点击"发送"时，通过用户配置的 API Key 发送到对应的 LLM 厂商。

3. 第三方服务

用户划选的文字和提问内容会被发送到用户自行选择的 LLM 厂商（DeepSeek、通义千问、智谱GLM、Kimi、OpenRouter 之一）。本扩展开发者不对这些第三方服务的隐私政策负责。请查阅对应厂商的隐私政策。

4. 数据安全

API Key 以明文形式存储在 chrome.storage.local。本扩展不通过任何方式向开发者或第三方服务器上传 API Key。

5. 联系方式

如有疑问，请通过 GitHub Issues 联系：github.com/mirtabesy821-source/Universal-Sub-Agent/issues
